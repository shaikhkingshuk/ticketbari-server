# TicketBari - Frontend

## Project Overview
TicketBari API is the backend for the TicketBari online ticket booking platform. It manages users, tickets, bookings, and payments. The API supports three user roles: User, Vendor, and Admin, with role-specific functionalities.

It handles all server-side logic, database operations, authentication, and integrates with Stripe for secure payments.

**Live Website:** [https://ticketbaribd.netlify.app/](https://ticketbaribd.netlify.app/)

## Key Features
- User management (create, update, roles, fraud marking)
- Ticket management (add, update, delete, verify, advertise)
- Booking system (book tickets, accept/reject, payment status)
- Stripe payment integration for secure online payments
- Vendor and admin dashboards with analytics
- Fetch advertised and latest tickets for the homepage


## Technologies & Libraries Used
- Node.js - JavaScript runtime
- Express - Web framework
- MongoDB - Database
- Stripe - Payment gateway integration
- dotenv - Environment variable management
- cors - Cross-Origin Resource Sharing

- **React Countdown** - Countdown timers  
- **React Icons** - UI icons
